git init . - dá um init
git add - adiciona ao repositório
git commit -m "frase" - cria commit com frase
git log - ve histórico de commits
git add . - manda todos os arquivos pro repositório
git reset (nome do arquivo.) - cancela o arquivo que foi mandado pro repositório sem dar commit


				###BRANCH###
git checkout -b (nome da branch que vc quiser) - cria uma branch (-b) é pra quando você ainda n tem uma branch
git branch - mostra todas as branchs do projeto
git checkout main - volta pra main	
git merge (traz arquivos da branch alterados) - quando vc está na branch main e traz informações da secundaria
git branch -d (nome da branch) - deleta a git secundaria

git remote add origin https://github.com/LuanRAr/comandosGit.git
$ git push -u origin main  -> manda pro GitHub

git clone (link que dá fork) -> pega arquivos que deseja trabalhar de outra pessoa
